### Soulmate

## 1. Clone this repo

```
git clone git@github.com:Soulmate-Lights/soulmate-example.git
cd soulmate-example
```

## 2. Installation:
```
./install
```

## 3. Customize your code:
Edit `main/main.cpp`.

## 4. Plug in your ESP32
Your LEDs should be wired like so:

- Ground -> Ground
- DATA -> pin 18
- CLOCK -> pin 23 (if using APA102)
- 5V -> 5V

## 4. Compile and upload:
```
./run
```
